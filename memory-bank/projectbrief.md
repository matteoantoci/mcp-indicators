# Project Brief

This document serves as the foundational document for the project, outlining its core requirements and goals.

## Project Name: mcp-indicators

## Core Requirements:
- Provide a set of technical indicators for financial data.
- Implement indicators such as SMA, EMA, RSI, MACD, Bollinger Bands, Stochastic Oscillator, ATR, Ichimoku Cloud, and Volume Profile.
- Make these indicators available as MCP tools.

## Goals:
- Create a reliable and efficient library for technical indicator calculations.
- Integrate with financial data sources (via other MCP servers) to fetch data for calculations.
- Provide clear and well-documented tools for users to access the indicators.
- Ensure the indicators are accurate and performant.
